# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within KubePilot, please DO NOT open a public issue. 
Instead, send an email to the core maintainers. We will respond within 48 hours to triage the issue.
